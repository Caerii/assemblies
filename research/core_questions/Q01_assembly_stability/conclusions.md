# Conclusions

## Defensible Conclusion

In the tested recurrent stim+self regimes implemented here, assemblies can form
autonomous patterns that persist far above chance after stimulus removal, and
that persistence is substantially stronger than the stim-only control.

## Current Boundaries

- This is an empirical conclusion about documented parameter regimes.
- It is not yet a general proof for arbitrary network settings.
- Multi-assembly interference and long-horizon drift remain separate questions.

## Next Steps

- connect this question to a formal claims document
- test stability under multiple competing assemblies
- derive a sharper null model than random overlap alone
