# Examples

- **01_basic_assembly_calculus.py** — Brain, project, merge, overlap (CPU, no GPU).  
  Run: `uv run python examples/01_basic_assembly_calculus.py` (from repo root) or `python examples/01_basic_assembly_calculus.py` after `pip install neural-assemblies`.

## Notebooks

- **notebooks/01_basic_assembly_calculus.ipynb** — Same example as above in notebook form.  
  From repo root: `uv run jupyter notebook` (or JupyterLab / VS Code), then open `examples/notebooks/01_basic_assembly_calculus.ipynb`.  
  If Jupyter isn’t installed: `uv add jupyter` or `pip install jupyter`.
